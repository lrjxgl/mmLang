# Api约定
 Api采用ResTApi风格，以模块化应用区分，index为基础应用,其他如forum论坛 
 每个应用可包含多个目录，如forum模块：  
 forum/index/forum,url:forum/index  
 forum/admin/forum,url:admin/forum/index  
 forum/shop/forum,url:shop/forum/index
